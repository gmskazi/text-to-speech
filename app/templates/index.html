How to use this tool

  1. Select the number of speakers and set your output filename.
  2. For one speaker, choose a language, voice, and speed rate, then paste your text in the single text box.
  3. For multiple speakers, choose a language, set each speaker voice/speed rate, and format dialogue as lines like A: Hi, How are you? and B: I'm fine, thank you.. Unlabeled lines continue with the last speaker.
  4. Click Generate MP3 to create and download the audio file.
