How to Use the Web App

  1. Open /.
  2. Set Speaker Count (1 to 4).
  3. Set Output Filename (auto-sanitized, .mp3 enforced).
  4. Single speaker: choose language/voice/rate, paste text, and set Natural Mode as needed.
  5. Dialogue mode: choose language, configure each speaker voice/rate, then enter dialogue lines.
  6. Click Generate MP3 to download.
